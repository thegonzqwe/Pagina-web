CURSO DISEÑO WEB

¿Qué es el diseño web? El diseño web implica trabajo relacionado con el layout y diseño de páginas online, así como la producción de contenido, aunque generalmente se aplica a la creación de sitios web.

En este caso, los diseñadores web crean las páginas utilizando lenguajes de marcado como HTML o XML.
